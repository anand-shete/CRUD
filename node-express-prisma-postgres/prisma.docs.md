Prisma is an ORM (Object Relational Mapper) for SQL databases like PostgreSQL, MySQL, SQLite, SQL Server, and CockroachDB.

# Components

Prisma ORM is an open-source next-generation ORM. It consists of the following ecosysteme:

1. Prisma Client: Auto-generated and type-safe query builder for Node.js & TypeScript
2. Prisma Migrate: Migration system
3. Prisma Studio: GUI to view and edit data in your database
